# Description
 This is an excel automation program developed for private use. 
###### How to use ?

Name of input excel file is not important you just need to set the name of worksheet **Sheet1**

Columns' order must be as follows.
- SollRückmeldetermin Leitstand
- Konstruktionstermin Soll
- Dokument
- Beschreibung
- BB-Nummer
- Dokumentstatus
- Auftragsphase
- BB Beschreibung
- KSW-Status
- Dokumentennummer Maßnahme
- Sachbearbeiter
